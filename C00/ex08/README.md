- Your task : Create a function that displays all unique combinations of n distinct digits in as-cending order, without repetition 
- The value of n will be such that: 0 < n < 10.
- Example output for n = 2:
    ```
    $>./a.out | cat -e
    01, 02, 03, ..., 09, 12, ..., 79, 89$>
    ```
- The function should be prototyped as follows: `void   ft_print_combn(int n);`
- Compiles with the following flags: `-Wall -Wextra -Werror`
