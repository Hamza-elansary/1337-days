- Your task : Create a function that displays all different combinations of two distinct two-digits numbers (XX XX) between 00 and 99, listed in ascending order.

- Expected output :
```
    $>./a.out | cat -e
    00 01, 00 02, 00 03, 00 04, 00 05, ..., 00 99, 01 02, ..., 97 99, 98 99$>
```
- The function should be prototyped as follows: `void ft_print_comb2(void);`
- Compiles with the following flags: `-Wall -Wextra -Werror`
